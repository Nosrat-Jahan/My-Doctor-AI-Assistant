# 🩺 My Doctor AI Assistant (v9.9.9)

### 📂 Project Archive: Created in February 2025
This repository contains a legacy build of an intelligent medical symptom checker. Developed as an exploration into the intersection of **Machine Learning (NLP)** and **Healthcare**, it utilizes a Multinomial Naive Bayes classifier to provide preliminary diagnostic insights.

---

## 🚀 Key Features
- **Naive Bayes Classification:** Maps natural language symptom descriptions to potential conditions.
- **Dynamic Database:** Integrated JSON-based store for conditions, medication, and dosage logic.
- **Modern UI:** Responsive frontend designed for clarity and professional clinical feel.
- **Version:** 9.9.9 (Original Legacy Build).

## 🛠️ Technical Stack
- **Backend:** Python / Flask
- **Machine Learning:** Scikit-Learn (CountVectorizer, MultinomialNB)
- **Frontend:** HTML5, CSS3 (Modern UI patterns), JavaScript (Async/Fetch)
